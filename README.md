A top-down driving game built in Unity (C#) where the player must avoid obstacles and survive as long as possible.

## Obstacles:
 ```
Pylons - Stun players for 1 second
Cars - Destroy the players car
 ```
## Game Layout:
```
4-lane street, going 2 ways, with pylons and cars spawning across multiple lanes.
-------------------------------------------------------------------------------------------------------------------------
>>  <-- Pylon                                        <-- Pylon                                         <-- Pylon       >>
>>          <-- Car                                                                         <-- Car                    >>                                                                  
>>                             <-- Car                             <-- Car                                             >>
<< <-- Pylon                                         <-- Pylon                                         <-- Pylon       <<
<<                    Car -->                              Car -->                                                     <<
<<                                                                                       Car -->                       <<
<< <-- Pylon                                         <-- Pylon                                         <-- Pylon       <<
-------------------------------------------------------------------------------------------------------------------------
```
